# JQ package 

Package **JQ** - lisp ([JSCL][jscl]) wrap for JQuery library.

It is intended for use in the [Moren][moren] environment.

## Status

`Development`

## Compilation

```lisp
   (lores:qload :jq :storejs t)
   (lores:modlink :jq "any-name.js")
```
